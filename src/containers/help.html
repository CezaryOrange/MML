Mapping My Life - Aide Utilisateur

Données utilisées

Elles sont récupérées par le connecteur « Orange Mobile – Appels & Géoloc » que vous devez activer au préalable.

Deux types de données sont affichés dans cette application :

  1. les localisations au moment des appels et SMS émis ou reçus depuis votre mobile ;
  2. les localisations toutes les demi-heures, lorsque vous avez explicitement accepté cette option dans le connecteur.

Les coordonnées de cette localisation sont celles de l’antenne radio sur laquelle votre mobile était connecté.

Ces données  sont mises à jour sur les serveurs d’Orange :

-          toutes les 2 semaines pour les données de type 1 ;

-          toutes les 30 minutes pour celles de type 2.

Elles sont téléchargées par le connecteur dans votre Cozy automatiquement une fois par semaine, ou manuellement à tout moment en cliquant sur le lien « SYNCHRONISER MAINTENANT » de l’onglet de synchronisation du connecteur.

Carte générale

Dans chacun des onglets, un fond de carte est présenté.

Vous pouvez naviguer dans cette carte

-          en la faisant glisser à l’aide de la souris : utiliser pour cela le bouton de gauche et le gardant appuyé ;

-          en zoomant sur la zone, soit à l’aide des boutons « + / - »  en haut à gauche, soit grâce à la molette de votre souris.

Onglet « Home »

Cette carte affiche l’ensemble des endroits où votre mobile a été localisé pour une période donnée.

Sous la carte, une timeline est affichée dans laquelle sont représentés :

-          dans la ligne « Geoloc », les points où votre mobile a été localisé à la demi-heure (données de type 2) ;

-          dans la ligne « PhoneCommunication », les points où votre mobile a émis ou reçu des appels ou des SMS (données de type 1).

Par défaut, cette timeline affiche les deux derniers jours, mais vous pouvez modifier la période observée à l’aide des boutons verts situés à sa droite :

-          « + / - » allonge ou restreint la période ;

-          « < / > » déplace la période.

En passant le pointeur de souris sur chaque point dans la timeline, un pop-up s’affiche avec la position et l’heure, ainsi que le type d’usage (appel ou SMS) pour la ligne « Phone Communication ». En cliquant sur un de ces points, la carte se déplace pour se centrer sur lui.

Lorsque le mobile a été localisé plusieurs fois à un endroit, une icône sur la carte indique la position et le nombre de fois où il a été vu par le réseau.

Les points sont regroupés avant l’affichage, selon leur proximité et le niveau de zoom que vous utilisez : si vous zoomez ou dé-zoomez, le niveau de détail est modifié.

En cliquant sur ces icônes représentées sur la carte, vous pouvez afficher la liste des localisations qui ont été regroupées, chacune item de la liste donnant le jour et l’heure où le mobile a été vu par le réseau (via les boutons « Affiche » / « Cache »).

Onglet « Favoris »

Cette carte affiche un regroupement de l’ensemble des points où votre mobile a été localisé durant toute la période d’observation.

Chaque icône représente un lieu que vous fréquentez souvent. Les 10 lieux les plus fréquentés (5 pour chaque type de données) sont représentés et peuvent être libellés comme Maison, Travail, Sport, Marché ou Autre (que vous pouvez préciser).

Pour libeller un lieu, cliquez sur son icône sur la carte, puis choisissez son type puis utilisez l’un des boutons « Ajouter » / « Modifier » / « Supprimer ».

Remarque : un bug en cours de traitement empêche temporairement l’affichage de la chaîne de caractères entrée dans « Autre ».

Onglet « Itineraire »

Cette carte affiche l’ensemble de vos trajets sur une journée.

Vous devez d’abord choisir une date en cliquant dans la zone de texte prévue à cet effet et en la sélectionnant dans le calendrier affiché, puis cliquer sur le bouton « Display ».

Le trajet affiché utilise l’ensemble des données disponibles pour la journée choisie (données de type 1 et de type 2) et une animation vous donne le sens de circulation.
